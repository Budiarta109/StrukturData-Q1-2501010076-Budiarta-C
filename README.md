# StrukturData-Q1-2501010076-Budiarta-C

## 1. 1. Karakteristik Memori dan Akses DataPerbedaan kecepatan akses antara Array dan Linked List terletak pada layout memori.Array (Kontinu): Elemen-elemen array disimpan dalam blok memori yang berurutan. Karena lokasinya berdampingan, komputer dapat menghitung alamat memori elemen ke-$i$ secara langsung menggunakan rumus:$$\text{Alamat} = \text{Alamat Dasar} + (i \times \text{Ukuran Data})$$Karena hanya membutuhkan satu kali perhitungan matematika sederhana, aksesnya bersifat $O(1)$ (Konstan).Singly Linked List (Non-Kontinu): Elemen (node) tersebar secara acak di memori. Satu-satunya cara mengetahui lokasi node ketiga adalah dengan mengunjungi node pertama, lalu mengikuti pointernya ke node kedua, baru sampai ke node ketiga. Proses penelusuran (traversing) ini harus dilakukan satu per satu, sehingga membutuhkan waktu $O(n)$ (Linear).
