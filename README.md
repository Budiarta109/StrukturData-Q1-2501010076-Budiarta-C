# StrukturData-Q1-2501010076-Budiarta-C

## 1. Kenapa Array Cepat 
($O(1)$) tapi Linked List Lambat ($O(n)$)?Bayangkan Array itu seperti perumahan townhouse. Rumah-rumahnya berurutan (kontinu) dan punya nomor yang pasti. Kalau kamu mau ke rumah nomor 5, kamu bisa langsung lari ke sana karena tahu persis lokasinya. Komputer tinggal hitung jaraknya, makanya instan ($O(1)$).Sedangkan Linked List itu seperti permainan mencari jejak (scavenger hunt). Rumah-rumahnya mencar (non-kontinu). Untuk sampai ke rumah ke-5, kamu harus ke rumah ke-1 dulu buat nanya alamat rumah ke-2, lalu ke rumah ke-2 buat nanya alamat rumah ke-3, dan seterusnya. Karena harus mampir satu-satu, jadinya lama ($O(n)$).
